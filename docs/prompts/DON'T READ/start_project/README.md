## For new projects

To bootstrap another repo, use the template at repository root or `bootstrap-ai-governed-docs.md` from the Handy org docs.

**Do not re-run bootstrap on MagicHandy** unless resetting the entire `docs/tasks/` and governance structure intentionally.
